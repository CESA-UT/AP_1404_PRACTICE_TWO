# WallNut

![تصویر اصلی WallNut](../../Assets/images/Plants/Wallnut.gif)


گیاه دفاعی بازی است و برای متوقف کردن زامبی‌ها استفاده می‌شود.

## وضعیت

الزامی


## مشخصات

| ویژگی | مقدار |
|---|---:|
| هزینه کاشت | ۵۰ Sun |
| HP | ۴۰۰۰ |
| cooldown کارت | ۳۰ ثانیه |
| نوع عملکرد | دفاعی |
| آسیب | ندارد |

## رفتار

- WallNut جلوی حرکت زامبی را می‌گیرد.
- زامبی وقتی به WallNut برسد، باید بایستد و شروع به خوردن آن کند.
- با کم شدن HP، WallNut باید در نهایت حذف شود.
- نمایش حالت آسیب‌دیده WallNut اختیاری است.
