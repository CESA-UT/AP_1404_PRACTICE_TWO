# راهنمای assetها

استفاده از assetهای داده‌شده برای بخش‌های اصلی بازی الزامی است.

## مسیرهای مهم

| بخش | مسیر |
|---|---|
| زمین بازی | `Assets/images/items/Frontyard.png` |
| کارت‌ها | `Assets/images/Cards/` |
| گیاه‌ها | `Assets/images/Plants/` |
| زامبی‌ها | `Assets/images/Zombies/` |
| آیتم‌ها | `Assets/images/items/` |
| صداها | `Assets/sounds/` |

## نکته

استفاده از انیمیشن کامل GIFها الزامی نیست.  
اما حداقل باید تصویر یا فریم مناسبی از asset مربوط به هر موجودیت نمایش داده شود.

## پیش‌نمایش تصویری

برای دیدن گالری تصویری assetها، فایل زیر را ببینید:

```text
docs/characters/Gallery.md
```
