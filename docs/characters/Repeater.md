# Repeater

![تصویر اصلی Repeater](../../Assets/images/Plants/Repeater.gif)


گیاه حمله‌ای اختیاری است که در هر نوبت دو Pea شلیک می‌کند.

## وضعیت

اختیاری


## مشخصات

| ویژگی | مقدار |
|---|---:|
| هزینه کاشت | ۲۰۰ Sun |
| HP | ۳۰۰ |
| cooldown کارت | ۷.۵ ثانیه |
| نوع حمله | شلیک دو Pea پشت سر هم |
| آسیب هر Pea | ۲۰ |
| فاصله زمانی بین شلیک‌ها | ۱.۵ ثانیه |

## رفتار

- مثل PeaShooter عمل می‌کند، اما در هر نوبت دو گلوله شلیک می‌کند.
- هر گلوله باید برخورد جداگانه داشته باشد.
- اگر گلوله اول زامبی را کشت، گلوله دوم باید بتواند به زامبی بعدی برخورد کند.
